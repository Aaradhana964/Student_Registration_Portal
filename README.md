# 🎓 Student Registration Portal

A Python-based Student Registration Portal that allows users to manage student records efficiently. The system enables student registration, profile image handling, data storage, and record management through a simple and user-friendly interface.

## 📖 Overview

This project is designed to simplify the process of managing student information. Users can register students, store their details, manage profile images, and retrieve student records when needed.

The project demonstrates the integration of:

* Python Programming
* Object-Oriented Programming (OOP)
* File Handling
* Image Processing
* Exception Handling
* Data Management

## 🚀 Features

* Register new students
* Store student information
* Upload and manage profile images
* Restore student profile images
* View student records
* User-friendly menu-driven interface
* File-based data storage
* Error handling and validation

## 🛠️ Technologies Used

* Python 3.x
* Object-Oriented Programming (OOP)
* File Handling
* Image Processing
* Exception Handling

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/Aaradhana964/Student_Registration_Portal.git
```

Move into the project directory:

```bash
cd Student_Registration_Portal
```

Run the project:

```bash
python Student_portal.py
```
